### 3. 大戶動向分析

**位置：** Tab 1 & Tab 3

**功能特色：**
- 估算大戶買進量
- 估算大戶賣出量
- 計算淨進出（買進 - 賣出）
- 判斷大戶狀態（進場/出場/觀望）

**算法說明：**
```python
# 簡化估算模型
avg_vol = 近 5 日平均成交量
大戶進出 = 成交量 > avg_vol × 1.5 的交易日
estimated_buy = 大戶進出量 × 30%
estimated_sell = 大戶進出量 × 20%
net_flow = estimated_buy - estimated_sell
```

**視覺化：** 長條圖顯示買進/賣出/淨進出分布

### 4. 成交量預測

**位置：** Tab 3 - 預測中心

**功能特色：**
- 使用線性回歸預測明日成交量
- 顯示預測值與今日比較增減%
- 趨勢圖展示實際 vs 預測

**算法說明：**
```python
# 取近 10 日成交量
recent_vol = df['volume'].tail(10).values

# 線性回歸求斜率
slope = np.polyfit(days, recent_vol, 1)[0]

# 預測明日
predicted_vol = recent_vol[-1] + slope
```

**準確度提示：** 此為簡化模型，實際應用可加入更多因子（如：市場情緒、外資動向等）

- 價格 vs 均線關係
- 波動率

if RSI > 70: up_prob += 15%
if 價格 > 1.05×SMA20: up_prob += 10%

### 8. 策略回測模擬

**位置：** Tab 4 - 策略回測

**功能特色：**
- 初始資金 10 萬元
- 交易訊號：MA+KD 組合策略
- 績效指標：勝率、累積報酬率、最大回撤
- 資產增長曲線圖
- 交易明細下載

**策略規則：**
```python
# 買入訊號
if 價格 > SMA20 AND KD 黃金交叉:
    signal = 1  # 買入

# 賣出訊號
if 價格 < SMA20 OR KD 死亡交叉:
    signal = -1  # 賣出
```

**績效指標：**
| 指標 | 說明 | 計算方式 |
|------|------|----------|
| 總交易次數 | 完整買賣回合數 | 賣出訊號執行次數 |
| 勝率 | 獲利交易佔比 | 獲利次數 / 總次數 × 100% |
| 累積報酬率 | 總收益百分比 | (最終資產 - 初始) / 初始 × 100% |
| 最大回撤 | 最大虧損幅度 | (最低點 - 最高點) / 最高點 × 100% |

## 🔧 常見問題

### Q1: 登入失敗
**A:** 檢查以下項目：
- 身分證字號、密碼是否正確
- 憑證路徑是否正確（注意反斜線 \）
- 憑證是否在有效期內
- 是否為交易時間（部分 API 僅限交易時段使用）

### Q2: 無法取得數據
**A:** 可能原因：
- 股票代號錯誤（應為 4 碼數字，如：2330）
- 非交易時段（無當日數據）
- API 連線不穩定

### Q3: 預測準確度不高
**A:** 本系統使用簡化模型，僅供參考：
- 成交量預測：僅用線性回歸，未考慮市場消息
- 漲跌停預測：僅用技術指標，未考慮基本面
- 建議搭配其他分析工具使用

### Q4: 大單監控無反應
**A:** 需連接 WebSocket 才能接收即時交易資料：
```python
sdk.init_realtime()
stock_client = sdk.marketdata.websocket_client.stock
stock_client.subscribe({'channel': 'trades', 'symbol': target_symbol})
```
