# 台股全方位戰情室系統

本專案提供三個不同介面的台股分析工具，皆整合富邦 API 進行資料抓取與分析。

## 📁 檔案說明

### 1. `taiwan_stock_streamlit.py` - Streamlit 網頁版
**適合對象：** 喜歡網頁介面、需要視覺化圖表的用戶

**功能特色：**
- ✅ 即時戰情監控（收盤價、預估量能、漲停空間）
- ✅ 技術分析儀表板（SMA 20、RSI 14、KD 指標）
- ✅ 策略回測系統（進出場訊號、勝率統計、資產曲線）
- ✅ 互動式 K 線圖表（Plotly 可縮放）
- ✅ 大單監控門檻設定

**執行方式：**
```bash
streamlit run taiwan_stock_streamlit.py
```

**使用流程：**
1. 在左側欄輸入富邦 API 憑證資訊
2. 點擊「連線富邦 API」
3. 輸入欲分析的股票代號（如：2330）
4. 切換不同分頁查看各項分析

---

### 2. `taiwan_stock_backtest.py` - 命令行回測版
**適合對象：** 偏好快速執行、只需要文字報告的用戶

**功能特色：**
- ✅ 歷史 K 線自動抓取（近 200 日）
- ✅ 技術指標計算（SMA、RSI、KD）
- ✅ 自動產生操作建議（偏多/偏空/觀望）
- ✅ 策略回測報告（勝率、累積報酬）

**執行方式：**
```bash
python taiwan_stock_backtest.py
```

**輸出範例：**
```
📊 數據最後日期：2025-04-14
   收盤價：98.5
   20MA: 96.20
   RSI: 58.32
   K: 72.15, D: 68.40

💡 明日操作建議:
   🟢 強勢偏多 (建議買進/持有)

📈 【策略回測報告 - 2330】
測試區間：2024-09-15 ~ 2025-04-14
總交易次數：12 次 | 模擬勝率：66.67% | 累積報酬：15.32%
```

---

### 3. `taiwan_stock_realtime.py` - WebSocket 即時監控版
**適合對象：** 需要即時大單警示的活躍交易者

**功能特色：**
- ✅ 即時成交資料監聽（WebSocket）
- ✅ 自訂大單門檻（張數）
- ✅ 即時警示通知（時間、價格、成交量）

**執行方式：**
```bash
python taiwan_stock_realtime.py
```

**輸出範例：**
```
🔥 [13:25:42] 【大單警示】
   股票代號：2330
   成交價格：98.5
   成交量：125 張
----------------------------------------
```

---

### 4. `taiwan_stock_monitor_gui.py` - PyQt5 桌面版（現有）
**適合對象：** 喜歡傳統桌面應用程式的用戶

**功能特色：**
- ✅ 完整 GUI 介面（PyQt5）
- ✅ 多分頁設計（即時監控、大戶動向、預測分析等）
- ⚠️ 目前為模擬模式，需連接真實 API

**執行方式：**
```bash
python taiwan_stock_monitor_gui.py
```

## 🔧 環境安裝

### 必要套件

```bash
# 基礎套件
pip install pandas plotly streamlit

# 富邦 API（需先下載 .whl 檔案）
pip install fubon_neo-<version>-cp37-abi3-win_amd64.whl

# PyQt5 桌面版（選用）
pip install PyQt5
```

### 版本相容性
- Python: 3.7+
- 作業系統：Windows（富邦 API 目前僅支援 Windows）
- 憑證：需具備富邦證券 API 憑證檔案（.p12）

---

## 📋 使用前準備

1. **申請富邦 API 帳號**
   - 前往 [富邦 FBS API](https://www.fbs.com.tw/) 申請
   - 取得身分證字號、登入密碼

2. **下載憑證檔案**
   - 從富邦官網下載 `.p12` 憑證檔
   - 記住憑證密碼

3. **修改程式碼中的憑證路徑**
   ```python
   CERT_PATH = r"D:\Users\YourName\Desktop\您的憑證.p12"
   ```

## 📊 技術指標說明

| 指標 | 參數 | 說明 |
|------|------|------|
| SMA | 20 日 | 20 日簡單移動平均線，判斷趨勢方向 |
| RSI | 14 日 | 相對強弱指標，>50 偏多，<50 偏空 |
| KD | 9,3,3 | 隨機指標，黃金交叉看多，死亡交叉看空 |

---

## 🎯 交易策略邏輯

### 買入訊號
- 收盤價 > 20MA **且**
- KD 黃金交叉（K 從下往上穿越 D）

### 賣出訊號
- 收盤價 < 20MA **或**
- KD 死亡交叉（K 從上往下穿越 D）
